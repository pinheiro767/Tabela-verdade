Regras essenciais da lógica proposicional

¬

Negação

¬p inverte o valor lógico de p.

V vira F; F vira V.

∧

Conjunção

p ∧ q é verdadeira apenas quando as duas proposições são verdadeiras.

Só V ∧ V = V.

∨

Disjunção inclusiva

p ∨ q é falsa apenas quando ambas são falsas.

Só F ∨ F = F.

⊻

Disjunção exclusiva

p ⊻ q é verdadeira quando apenas uma é verdadeira.

Valores diferentes = V.

→

Condicional

p → q é falsa somente quando p é verdadeira e q é falsa.

V → F = F.

↔

Bicondicional

p ↔ q é verdadeira quando p e q têm o mesmo valor.

Valores iguais = V.

Equivalências muito cobradas

Condicional: p → q ≡ ¬p ∨ q

Contrapositiva: p → q ≡ ¬q → ¬p

Negação da condicional: ¬(p → q) ≡ p ∧ ¬q

Leis de De Morgan: ¬(p ∧ q) ≡ ¬p ∨ ¬q

Leis de De Morgan: ¬(p ∨ q) ≡ ¬p ∧ ¬q

Bicondicional: p ↔ q ≡ (p → q) ∧ (q → p)
