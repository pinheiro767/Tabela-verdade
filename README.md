# Lógica de Concurso — PWA de Tabela-Verdade

Aplicativo PWA feito com HTML, CSS e JavaScript puro.

## Recursos

- Gerador de tabela-verdade com até 5 proposições.
- Conectivos: negação, conjunção, disjunção, exclusiva, condicional e bicondicional.
- Classificação automática em tautologia, contradição ou contingência.
- Questões autorais no estilo de concursos.
- Correção e explicação automática.
- Histórico e desempenho salvos no aparelho.
- Exportação da tabela em CSV.
- Tema claro e escuro.
- Uso offline.
- Instalação como aplicativo.

## Como publicar no GitHub Pages

1. Crie um repositório público no GitHub.
2. Envie todos os arquivos e a pasta `assets`.
3. Abra `Settings`.
4. Entre em `Pages`.
5. Em `Build and deployment`, selecione `Deploy from a branch`.
6. Escolha a branch `main` e a pasta `/root`.
7. Clique em `Save`.
8. Aguarde o GitHub exibir o endereço do aplicativo.
